# Tech Stack
- **Language:** SWIFT 5
- **IDE:** Xcode 14
- **iOS:** 16
- **Archetecture:** MVC
- **Frameworks:**  UIKit, Foundation, Realm
- **Layout:** From Figma, Interface Builder
- **GCD:** Yes 
- **Features:** 
  - URLSession
  - Saving data to Realm
  - Pull to refresh for Realm
  - DataModel
  - Rest API
  - Json decoding and ecoding
  - TableView
  - Custom fonts
  - Custom swipes
  - Delegates
  - Segues
  - Unwind
  - And many more
